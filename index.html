Quick Start & Support

Share this with table workers and officials.

How to Run a Mat

  • Open Control Panel and select the correct mat from the dropdown.
  • Set wrestler names in the ⚙️ drawer (Match Settings).
  • Choose the period length in the ⚡ drawer (Timer Settings).
  • Use the Start/Stop/Reset buttons to manage the clock.
  • Use the scoring buttons (E1, R2, T3, N2, N3, N4) to update the score.
  • Scoreboard pages will update live for each mat.

Common Setup

  • One laptop or tablet per mat running the Control Panel.
  • One display (TV/projector) per mat running the Scoreboard for that mat.
  • An extra device can run the Overview for the head table.

Troubleshooting

  • If Status shows “connecting…” for more than a few seconds, refresh the page.
  • If a scoreboard does not update, verify the correct mat number is selected in the Control Panel.
  • Check network/Wi-Fi if all mats disconnect at once.
